Infrared thermal temperature measuring device


"March 2023"

About the Project


This project is about making an Infrared thermal temperature measuring device from scratch. The components used to make the same are: Arduino Pro Mini, IR Temperature Sensor (MLX90614), OLED Display (128x64), Lithium Ion Battery (3.7V,2500mAh), Jumper Wires, Switch, Laser. The Arduino Pro Mini is a microcontroller board.
